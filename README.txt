Design files for asimple acrylic chip guard for use with the CRP2448 CNC
router.  The chip guards attach via magnets to mount plates on gantry carriages
and can be quickly removed and re-attached.

The project includes dxf drawing, gcode generation via py2gode and gcode files for cutting from a
12"x24" sheet of acrylic.

Design files: crp2448_chip_guard_left.dxf, crp2448_chip_guard_right.dxf

See operations.txt for notes and the the order of machining operations.
